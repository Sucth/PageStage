Un problème venu de l'intérieur

Pendant mon stage nous avons eu un problème venant de l'intérieur… Pour régler ce problème nous avons appliqué :

  • Analyse
  • Résolution
  • Améliorations

Analyse

Durant la quatrième semaine, j’ai eu un problème avec un membre de l'équipe. En effet, j’ai remarqué qu'une personne de l'équipe allait sur des sites d’argent pendant les heures de travail et autres sites, mais il utilisait son PC professionnel pour jouer. Ensuite, j’ai analysé les logs de GLPI pour voir quel logiciel avait été installé et, en effet, plusieurs logiciels de jeu étaient installés. Pour confirmer cela, j'ai analysé les logs de Trend qui renvoient toutes les analyses du PC et j’ai découvert que cette personne utilisait aussi des extensions Chrome pour jouer, regarder Netflix, etc.

Résolution

Pour résoudre ce problème, j’ai trois points à traiter :

    • Les sites WEB
    • Les applications
    • Les extensions

  • Les sites web

Pour traiter le problème des sites web qui ne devraient pas être utilisés pendant les heures de travail, j’ai utilisé la web réputation de Trend Micro pour bloquer les sites sur lesquels ils ne doivent pas aller (jeu d’argent, jeux, sites pornographiques, etc.).

  • Les applications

Pour régler le problème des applications installées, nous avons découvert que pour installer ces applications, cette personne utilisait son N2 pour pouvoir les installer. Nous avons donc supprimé son N2 et créé des règles sur les applications qui ne doivent pas être installées.

  • Les extensions

Pour régler le problème des extensions, nous avons d'abord essayé de les bloquer dans Trend Micro, mais nous n'avons pas réussi car les extensions ne sont pas considérées comme des pages web donc ne sont pas bloquables comme un site web. Nous avons donc décidé de passer par GPO pour bloquer les extensions de Chrome. Ce que nous avons fait est de bloquer toutes les extensions sur Chrome sauf Trend Micro pour continuer à avoir la protection de l’antivirus sur toutes les machines.

Améliorations

Pour améliorer le système et la sécurité, nous avons résolu ces problèmes car ils représentent une menace en termes de sécurité. Un logiciel malveillant installé avec un N2 qui parvient à s'introduire dans le système peut être dévastateur. Le premier point d'amélioration était de traquer les logs Winlogbeat (application qui renvoie des logs) qui sont envoyés au SOC, car dessus nous voyons une analyse complète du système. Avec cette application, nous sommes capables de détecter toute dérive et d'agir au plus vite.
